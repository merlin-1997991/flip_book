# Design System Documentation

## Brand Identity

This design system defines the visual language and styling rules for all web pages and digital products. Consistency in applying these guidelines ensures a cohesive user experience across all touchpoints.

---

## Color Palette

### Primary Colors

#### Fluorescence Turquoise (PC-1)
- **Base Color:** `#50dce1`
- **Usage:** Primary actions, brand accent, interactive elements
- **Accessibility:** Use with dark text for sufficient contrast

**Tints:**
- PC-1: `#50dce1` (Base)
- PC-1-80: `#73e3e7`
- PC-1-60: `#96eaed`
- PC-1-40: `#b9f1f3`
- PC-1-20: `#dcf8f9`

#### Dark Purple (PC-2)
- **Base Color:** `#32005a`
- **Usage:** Headers, footer backgrounds, primary text on light backgrounds
- **Accessibility:** Provides strong contrast with white text

**Tints:**
- PC-2: `#32005a` (Base)
- PC-2-80: `#5b337b`
- PC-2-60: `#84669c`
- PC-2-40: `#ad99bd`
- PC-2-20: `#d6ccde`

#### White (BG-1)
- **Base Color:** `#ffffff`
- **Usage:** Page backgrounds, card surfaces, text on dark backgrounds

#### Black (BG-2)
- **Base Color:** `#000000`
- **Usage:** Body text, icons, borders

**Opacity Variants:**
- BG-2: `#000000` (100%)
- BG-2-80: `#000000` at 90% opacity
- BG-2-65: `#000000` at 65% opacity
- BG-2-45: `#000000` at 45% opacity
- BG-2-25: `#000000` at 25% opacity
